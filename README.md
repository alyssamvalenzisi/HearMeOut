## General info
This project is Python Project created in the IDE Visual Studio Code.
It is a simple quiz that generates music genre recomendations, with a chance to get one of three artists and one song from that genre.
Uses TKinter for the quiz visual/interface, and uses Randomisation to to mix up the genre song/artist results.
The data set of questions, genres, artists and songs were manually created.
	
## Technologies
Project is created with:
* Visual Studio Code
	* Python as the coding environment
	* Tkinter to visualise the quiz in a window

## SetUp
1. Copy the Code
2. Open it in a Python Launcher or an IDE like Visual Studio Code (this the environment the code was created)
3. Run the Code to launch the quiz in an interactive interface
4. Have fun answering the questions in the quiz and seeing the Genre, Artist, and Song recommendation!
